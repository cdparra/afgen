========================
AFGENJAVA README
========================

AFGEN es un traductor de expresiones regulares a automatas finitos. Puede generar a
partir de una expresión regular que soporta las operaciones *,+,?,|, el Automata Finito
no Determinista, el Automata Finito Determinista y el Automata Finito Determinista Mínimo
utilizando las construcciones de thompson, el algoritmo de subconjuntos y de minimización. 

Para ejecutar el programa, debe tener instalada la máquina virtual de Java. Puede correr
programa haciendo doble click sobre el jar en un entorno de ventanas, o ejecutando el 
siguiente comando en una terminal: 

java -jar "afgenjava.jar" 

Para redistribuir el proyecto, comprimalo en un .zip, incluyendo el directorio lib y archivo
conf.xml
